En esta sección hablaremos de cada una de las formas con las que podemos agregar código CSS
a un documento HTML; Própiamente nos vamos a referir a los lugares en donde puede estar éste alojado.
Para ello hablaremos de tres maneras o ubicaciones en específico inline, interno y externo.



Inline


Se llama de esta forma cuando ponemos el código CSS dentro de la etiqueta HTML
que queremos estilizar. Se agrega dentro de la etiqueta la palabra Style = “” y
dentro de las comillas se ingresan cada una de las propiedades separadas por punto
y coma (;) entre ellas.

Externo


Este estilo es el mas recomendado, puesto que lo que se hace es crear un archivo .css aparte en
el que se alojará la totalidad del código que dará estilo a la página. De esta manera el documento
HTML únicamente alojará la estructura y código de este tipo y no se mezclará con los estilos.
